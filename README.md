# 📺 JJKC YouTube Algorithm Project

유튜브 알고리즘을 분석하고 다른 사람과 공유하는 서비스 프로젝트입니다.

## 📂 프로젝트 구조

- `prototype/`: 서비스의 주요 기능을 빠르게 확인해 볼 수 있는 프로토타입 (Next.js 기반)
- `plan.md`: 전체적인 서비스 기획 및 단계별 로드맵

## 🚀 프로토타입 실행 방법

현재 `prototype` 폴더에 구현된 데모 서비스를 실행하는 방법입니다.

### 1. 폴더 이동
```bash
cd prototype
```

### 2. 패키지 설치
```bash
npm install
```

### 3. 개발 서버 시작
```bash
npm run dev
```

### 4. 접속
브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하세요. 별도의 API 키 설정 없이도 미리 준비된 **Mock Data**를 통해 모든 기능을 바로 체험해 보실 수 있습니다.

---

## ✨ 주요 기능 체험
- **알고리즘 분석:** 내 시청 기록을 기반으로 한 관심사 분석 카드 생성
- **탐색 및 팔로우:** 다른 사람의 알고리즘 피드 구경 및 팔로우
- **알고리즘 비교:** 두 사용자의 알고리즘 차이를 한눈에 비교
